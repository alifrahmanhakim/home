Tutorial Workflow Zoho Projects

Langkah 1 dari 17

Langkah 1: Akses Zoho Project dan Menu "Task"

Untuk memulai, Anda akan melihat dasbor utama Zoho Project. Cari Panel Navigasi utama.

Ini adalah tampilan awal saat Anda masuk ke Zoho Projects.

☰ Home

Akses Panel Navigasi utama yang dilambangkan dengan ikon tiga garis horizontal di sudut kiri atas antarmuka.

Aksi: Klik pada ikon tiga garis (☰) untuk membuka Panel Navigasi.

Langkah 2: Mengakses Project Terbaru

Setelah membuka Panel Navigasi, pilih opsi 'Recent Projects'.

Home
Reports
Projects
Recent Projects

Daftar Project yang baru saja diakses akan muncul di bawah 'Recent Projects'.

CASR 91
CASR PART XX CORSIA
CASR 45

Aksi: Klik pada 'Recent Projects', lalu pilih nama Project yang relevan (misalnya, "CASR 91").

Langkah 3: Dasbor Project & Akses Menu "Tasks"

Anda akan diarahkan ke Menu Dashboard Project yang dipilih. Dari sini, navigasikan ke menu 'Tasks'.

Ini adalah dasbor spesifik untuk Project "STD-18 CASR 91". Anda dapat melihat ringkasan status Task.

Dashboard Tasks Reports Documents PROLEGKUM 2025

Aksi: Klik pada menu 'Tasks' untuk mengakses dan mengelola seluruh daftar Task Project.

Langkah 5: Menambahkan Daftar Task (Task List) Baru

Untuk menambahkan 'Task List' baru, gulir ke bagian paling bawah halaman daftar Task yang sudah ada. Klik pada area input yang tersedia.

Di bawah daftar Task yang ada, Anda akan menemukan opsi ini.

... (daftar Task yang sudah ada) ...

Create Task List Above
Create Task List Below

Aksi: Pilih "Create Task List Below" (atau "Above"), lalu masukkan Judul yang deskriptif untuk 'Task List' baru tersebut.

Langkah 6: Melihat Detail Task List & Menambahkan Task

Untuk menambahkan Task individual ke dalam 'Task List' yang sudah ada, pilih kolom 'View' pada 'Task List' yang bersangkutan.

Klik 'View' pada Task List "Gap Analisis..." untuk melihat detailnya.

# Gap Analisis (Matrix Annex vs Existing CASR) View

Setelah masuk ke detail Task List:

Add Task

Aksi: Klik tombol "Add Task", lalu input Judul Task. Status awal akan menjadi "Open".

Langkah 7: Menambahkan Subtask

Untuk menambahkan Subtask pada Task individual, klik opsi 'View' pada Task spesifik tersebut. Kemudian gulir ke bawah hingga menemukan bagian 'Subtasks'.

Misalnya, Anda melihat detail Task "Matriks".

Task: Matriks

Status: Open

Comments Subtasks Documents Add Subtask

Aksi: Masukkan uraian atau deskripsi untuk Subtask pada kolom yang disediakan ("Add Subtask").

Langkah 8: Menyisipkan Dokumen (Bagian 1)

Dokumen pendukung dapat disisipkan pada Task individual. Klik "View" pada salah satu "Task name" dalam "Task list".

Anda berada di daftar Task. Pilih Task yang ingin ditambahkan dokumen, misalnya "Matriks", lalu klik "View".

C91-T134 Matriks View Open

Aksi: Setelah mengklik "View" pada Task "Matriks", Anda akan masuk ke detail Task tersebut. Gulir ke bawah menuju pilihan "Documents".

Langkah 9: Menyisipkan Dokumen (Bagian 2)

Setelah masuk ke detail Task dan menggulir ke bawah, Anda akan menemukan tab "Documents".

Di dalam detail Task "Matriks":

Comments Subtasks Documents Invoices

No documents associated

Drop or attach relevant documents here.

Attach Files

Aksi: Klik tombol "Attach Files" untuk menyisipkan dokumen pendukung.

Langkah 12: Menentukan Tanggal Pelaksanaan (Bagian 1)

Pilih Task dalam tabel "Task List". Untuk menentukan tanggal mulai, klik pada kolom "Start Date" yang berada di baris "Task Name" yang akan diatur.

Anda berada di detail Task List "Gap Analisis...".

ID Task Name Status Start Date Due Date
C91-T120 Proses Drafting Internal Annex 2 In Progress mm-dd-yyyy

Aksi: Klik pada kolom "Start Date" (mm-dd-yyyy) untuk Task "Proses Drafting Internal Annex 2".

Langkah 13: Menentukan Tanggal Pelaksanaan (Bagian 2 - Start Date)

Sebuah kalender akan muncul. Tentukan tanggal dimulainya "Task" tersebut.

Pilih tanggal dari kalender.

Juni 2025

MINSENSELRABKAMJUMSAB 2627282930311 2345678

Aksi: Pilih tanggal mulai yang diinginkan dari kalender.

Langkah 14: Menentukan Tanggal Pelaksanaan (Bagian 3 - Due Date)

Dengan metode yang serupa, pilih kolom "Due Date" untuk menetapkan tenggat waktu penyelesaian Task. Kolom "Duration" akan terisi otomatis.

Setelah Start Date dan Due Date dipilih:

Start Date

01-01-2025

Due Date

07-31-2025

Duration

212 days

Klik pada kolom "Due Date" (07-31-2025) untuk memilih tanggal dari kalender.

Aksi: Klik kolom "Due Date" dan pilih tanggal akhir dari kalender. Durasi akan dihitung otomatis.

Langkah 15: Mengisi Kolom "Status" (Bagian 1)

Kolom "Status" berfungsi untuk melacak perkembangan terkini dari suatu Task. Pada menu Task, klik pada kolom "Status" di baris Task yang relevan.

Anda berada di detail Task List.

IDTask NameCompletion %Status
C91-T134Matriks0% Open
C91-T129CCEFOD100% Closed

Aksi: Klik pada kolom "Status" untuk Task "Matriks" (yang saat ini "Open").

Langkah 16: Mengisi Kolom "Status" (Bagian 2 - Pilih Status)

Pilihan status yang tersedia umumnya meliputi "Open", "In Progress", dan "Closed". Pilihlah status yang paling sesuai.

Setelah mengklik kolom status untuk Task "Matriks":

Open
In Progress
In Review
Closed
On Hold

Aksi: Pilih status yang sesuai, misalnya "Closed".
